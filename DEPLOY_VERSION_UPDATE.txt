┌───────────────────────────────────────────────────────────────┐

  Find lines 3-4 and update:

  const CACHE_VERSION = 'v4.1'                    ← Increment this
  const BUILD_TIMESTAMP = '2025-01-26T12:00:00Z'  ← Current timestamp


┌───────────────────────────────────────────────────────────────┐
│ STEP 2: Update Manifest (public/manifest.json)               │
└───────────────────────────────────────────────────────────────┘

  Find lines 4-6 and update:

  "version": "4.1.0",          ← Match service worker version
  "start_url": "/?v=4.1.0",    ← Update query parameter


┌───────────────────────────────────────────────────────────────┐
│ STEP 3: Commit & Deploy                                       │
└───────────────────────────────────────────────────────────────┘

  git add public/sw.js public/manifest.json
  git commit -m "chore: bump version to v4.2.0"
  git push


┌───────────────────────────────────────────────────────────────┐
│ STEP 4: Verify (after 5-10 minutes)                          │
└───────────────────────────────────────────────────────────────┘

  ✓ Open app on phone
  ✓ Wait 5 minutes
  ✓ See "Update Available" prompt
  ✓ Click "Update Now"
  ✓ Verify new version loaded
